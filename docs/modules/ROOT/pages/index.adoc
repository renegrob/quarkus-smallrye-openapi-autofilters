== Installation

If you want to use this extension, you need to add the `io.quarkiverse.smallrye-openapi-extras:quarkus-smallrye-openapi-extras` extension first.
In your `pom.xml` file, add:

[source,xml]
----
<dependency>
    <groupId>io.quarkiverse.smallrye-openapi-extras</groupId>
    <artifactId>quarkus-smallrye-openapi-extras</artifactId>
</dependency>
----
